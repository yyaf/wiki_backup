# datacom 实验环境搭建

> 实验环境：`win 10 19043.1387`。
>> 注意事项（坑）：关闭 `hyper-V`。
>>> 1. 打开`启用或关闭Windows功能`,取消勾选`hyper-V`。
>>> 2. 以管理员身份打开`CMD`，执行`bcdedit /set hypervisorlaunchtype off`。

1. 安装`WinPcap`。
2. 安装`Wireshark`。
3. 安装`VirtualBox-5.2.x`。
4. 安装`eNSP`。
